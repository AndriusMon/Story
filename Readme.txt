Commands used to work with GIT:
1. $ git init       -> initializes GIT for a project
2. $ git status     -> shows current state of each file
3. $ git add <file name>    -> adds file into working space
4. $ git commit -m <"Comment of a commit">  -> moves file into GIT repository
5. $ git log        -> shows history of commits

6. $ git remote add origin https://github.com/<UserName>/<RepositoryName>.git -> creates link with remote repository
7. $ git branch -M main -> creates main branch
8. $ git push -u origin main -> pushes project from local repository to remote repository
